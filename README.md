# vim-far
A Far Manager like Color Scheme (Vim/gVim).
Used [cSyntaxAfter][vimorglink] for highlighting operators, brackets. Download and copy `colors` dir to `{path/to/vim}/vimfiles` and set `:colorscheme far`
